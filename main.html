Question-1: What are the different ways to select an element in the DOM?

1.getElementById--Selects an element by its id
2.getElementByClassName--- select elements by their class name return html collection
3.getElementByTagName--- select elements by their tag name return html collection
4.querySelector---- select the first matching element using a Css selector.
5.querySelector--- selects all matching elements using a css selector returns a nodelist.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML: Gets or Sets the html content inside an element interpreting and rendering any html tags

innerText: Gets or Sets the visible tesxt inside an element, ignoring hidden elements and formating text.

textContent: Gets or Sets the raw text content inside an element, including hidden text,without interpreting html.

Question-3: What is event delegation in the DOM?

Event Delegation is a technique in the dom where a single event listener is added to a parent element to handle event on its childd element event if they are dynamically added later.

Question-4: What is event bubbling in the DOM?

Event Bubbling is a phase in the DOM event propagation procces where an event starts from target element and moves upward through its ancestors to the root. Default Behavior of most events. Usefull for event delegation attaching a listener to a parent instead of multiple children.
